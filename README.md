# Lumia-ESP32
This demo project for the WT32-SC01 module.

These are the current features
### Screens
- Boot Screen
- Lock Screen
- Start Menu
- Apps List
### Apps
- Settings
- LED Control App (needs ext. WS2812 leds)
### Modules
- Status Bar
- Notification Panel
- Navigation Bar (needs ext Motor for vibration)
- Alert Dialog
- Keyboard (not tested/used)
